What the Avro PHP library is
============================

A library for using [Avro](http://avro.apache.org/) with PHP.

Requirements
------------
 * PHP 7.2+
 * On 32-bit platforms, the [GMP PHP extension](http://php.net/gmp)

Getting started
---------------
```
$ composer require dealroom/avro
```
